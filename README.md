# 📖 Budget app <a name="about-project"></a>

**Budget app** is a mobile web application where users can manage their budget: one has a list of transactions associated with a category, so that one can see how much money one has spent and on what.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:dlwub/budget-app.git
```

### Install

Install this project with:

```sh
  cd my-project
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  rspec spec/models/model_name_spec.rb
```
